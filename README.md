# Intrusion Detection System using CIC-IDS2017

## Overview
This project implements a **machine learning–based intrusion detection system (IDS)** to classify network traffic as benign or malicious using the **CIC-IDS2017 dataset**. It demonstrates an end-to-end ML workflow applied to a real-world cybersecurity problem.

---

## What This Project Demonstrates
- Applied machine learning for cybersecurity use cases  
- End-to-end pipeline: data preprocessing → modeling → evaluation  
- Handling large, imbalanced network traffic datasets  
- Model comparison and performance analysis  

---

## Models & Techniques
- Logistic Regression  
- Decision Tree  
- Random Forest  
- Support Vector Machine (SVM)
- PCA Principal Component Analysis 
- Feature scaling and preprocessing  

---

## Tech Stack
- Python  
- scikit-learn  
- pandas, NumPy  
- matplotlib  
- Jupyter Notebook  

---

## Dataset
- CIC-IDS2017 (realistic network intrusion dataset)  
- Labeled benign and malicious network traffic  
- High-dimensional network flow features  

---

## Evaluation Metrics
- Accuracy  
- Precision  
- Recall  
- F1-score  
- Confusion Matrix  

---

## Key Outcomes
- Successfully classified malicious vs. benign traffic  
- Tree-based models captured complex feature interactions effectively  
- Demonstrated the importance of preprocessing in intrusion detection tasks
